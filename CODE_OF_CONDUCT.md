# Código de Conduta

## Compromisso

Este projeto deve ser um ambiente respeitoso, seguro e produtivo para colaboração.

Todos os participantes devem agir com profissionalismo, empatia e responsabilidade.

## Comportamentos esperados

- Tratar outras pessoas com respeito.
- Fazer críticas técnicas de forma objetiva.
- Explicar decisões e discordâncias com clareza.
- Aceitar feedback quando houver evidência técnica.
- Manter foco no problema e no usuário final.

## Comportamentos não aceitos

- Ataques pessoais, insultos ou intimidação.
- Discriminação de qualquer tipo.
- Exposição de dados pessoais ou sensíveis.
- Comentários ofensivos ou humilhantes.
- Pressão para aceitar mudanças sem revisão.

## Aplicação

Contribuições que violem este código podem ser recusadas, revertidas ou bloqueadas.

Mantenedores podem solicitar ajustes de tom, escopo ou conteúdo antes de aceitar uma contribuição.

## Reporte

Se você identificar uma violação, reporte ao mantenedor do projeto com:

- descrição objetiva do ocorrido;
- links, prints ou referências quando existirem;
- impacto percebido;
- sugestão de encaminhamento, se houver.
